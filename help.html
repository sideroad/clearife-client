What is GTD

デビッド・アレン（David Allen）が提唱したGTD（Getting Things Done）とは個人用のワークフロー管理システムです。 心理的な負担を減らすことを主眼とし、簡単な5つのステップを実行することによって成し遂げたいことを現実にするメソッドです。個人の生産性を向上させることができるとされ、LifeHack（ライフハック）の1つです。基本はなすべき仕事のリストを何かに記録しておくことで、頭の中からなすべき仕事のことを追い出すものです。これで頭の中はすべき仕事全部を覚えなくてもよくなりすっきりとし、リストに基づき実際の仕事をこなすことに集中できます。

What is clearife?

GTDを円滑に進めるためのシステムです。
簡素な画面とユーザビリティに配慮した操作性の実現を目指しています。

Getting Started

タスクの作成

現在抱えている仕事、プライベートの予定を全て洗い出します。
ただし、２分以内で実行できるものは実行し、残りのタスクを作成していきます。
Newと書いてあるタスクを押すと、タスクを作成できます。
タスク名、その詳細、開始日、期日を入力し(※1)、「Create」ボタンを押します。
「Current」または「Future」のボックスにタスクが生成されます。※2

※1詳細、開始日、期日は省略できます。
※2開始日が今日の場合「Todo」レーンに生成されます。
　 開始日が明日の場合「Tomorrow」レーンに生成されます。
　 開始日が明後日以降の場合「Definite」レーンに生成されます。
　 開始日が決まっていない場合「Someday」レーンに生成されます。

タスクの参照

タスクをダブルクリックすると詳細情報が参照できます。
また、詳細情報の下に表示されているボタンで各種操作が出来ます。
変更 タスクの内容を変更できます。
なお、Done,Resultsのタスクに関しては終了日の変更も出来ます。
削除 タスクを削除できます。削除されたタスクはゴミ箱(Trash)に移動します。
複製 タスクを同じレーンに複製します。
委譲 タスクを友達、若しくはチームのメンバーに委任します。
チームに属さないタスクは友達のみ委任できます。 チームに属すタスクはチームメンバーのみ委任できます。
Eメール タスクの内容をEメールで送信できます。
アドレスの入力欄の右横に表示されている「＠」アイコンをクリックすると、
アドレス帳を開くことが出来ます。アドレス帳の操作方法はこちら
ゴミ箱(Trash)のタスクは上記ボタンの代わりに以下のボタンが表示されます。
元に戻す タスクを元のレーンに移動します。
抹消 タスクを抹消します。抹消されたタスクは完全に削除されます。

ボックスについて

タスクは「Future」,「Current」,「Results」,「Trash」のいずれかのボックスに格納されます。

ボックス名 レーン名 備考
Future Someday いつ行うか決まっていないタスク
Definite 開始日が明後日以降のタスク
Tomorrow 開始日が明日のタスク
Current Todo 今日実行すべきタスク
タスクを着手する場合、ドラッグ＆ドロップで「Doing」に移動します。
Doing 現在実行中のタスク
タスクが完了した場合、ドラッグ＆ドロップで「Done」に移動します。
Done 今日完了したタスク
完了した日から一日経つと自動的に「Results」に格納されます。
Results Results 昨日以前に完了したタスク
Routine Routine 定期的に発生するタスクを管理します。
詳細はこちらを参照してください。
Trash Trash 削除されたタスクを格納します。
左側のメニューに表示されているシュレッダーのアイコンをクリックすると、
ゴミ箱の全タスクを抹消し、空にします。

ルーチンについて

ルーチンでは定期的に発生するタスクを登録、変更、削除できます。

チームについて

チームで仕事をこなしたり、タスクを友達と共有したり、単にタスクをカテゴライズしたい場合
チームを登録することで、円滑に共有を行えます。

アドレス帳について

タスクをEメールに送信する際にアドレス帳から選択することが出来ます。
アドレス帳を操作するボタンは以下の通りです。
選択する 一つ以上のアドレスを選択し、「選択する」ボタンを押すと、選択したメールアドレスが宛先に設定されます。
新規登録 新規アドレスを登録します。
変更する 一つ以上のアドレスを選択し、「変更する」ボタンをクリックすると、選択したアドレスの内容を変更できます。
削除する 一つ以上のアドレスを選択し、「削除する」ボタンをクリックすると、選択したアドレスを削除できます。
